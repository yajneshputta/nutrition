## 1. Overview

This app allows you to upload a picture of food and get an approximate nutritional analysis for the items present in the image. The backend sends the image to the Gemini vision model, receives a Markdown table describing the nutritional values, and displays it nicely on the frontend.

> âš ï¸ The nutrition values are **estimates** generated by an AI model and are **not** medically accurate. Use them only for rough reference, not for health-critical decisions.

---

## 2. Features

- Upload a **food image** (JPG/PNG).
- Sends the image to **Google Gemini (vision)** using the `google-generativeai` Python SDK.
- Gets a **Markdown table** with:
  - Food item name  
  - Calories  
  - Protein  
  - Fat  
  - Carbohydrates  
  - Fiber  
- Frontend JavaScript converts the Markdown table into a **styled HTML table**.
- Shows:
  - A preview of the **uploaded image**.
  - A **loading indicator** while analysis is running.
  - The **nutrition table** with estimated values.

## 5. How the Code Works

### 5.1 `app.py` (Flask Backend)

**Imports:**
- `Flask`, `render_template`, `request`
- `os`, `mimetypes`
- `google.generativeai as genai`

**Setup:**
```python
app = Flask(__name__)
app.config['UPLOAD_FOLDER'] = 'static/uploads'
genai.configure(api_key=os.getenv("GOOGLE_API_KEY"))
```

**Core function:** `get_nutrition_info(image_path)`  
- Reads image as bytes  
- Determines MIME type using `mimetypes.guess_type`  
- Creates Gemini model instance  
- Sends prompt + image to `model.generate_content()`  
- Returns Markdown text  

**Route:**
- **GET:** shows upload UI  
- **POST:** saves file â†’ runs model â†’ displays results
